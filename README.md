# MejoraCRM

CRM desarrollado por **Mejora Continua®** para gestión de clientes, productos e interacciones.

## Stack

- **Frontend:** React 18 + TypeScript + Vite
- **UI:** Tailwind CSS + shadcn/ui
- **Backend:** Supabase (Auth + PostgreSQL)
- **Gráficos:** Recharts

## Desarrollo

```bash
bun install
bun dev
```
