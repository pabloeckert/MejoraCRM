# Project Memory

## Core
CRM forestal/agro "Mejora Continua" para PyMEs familiares. 1-5 usuarios. Roles: vendedor, supervisor, admin.
Identidad: Azul #495F93 (primary), Dorado #E5C34B (accent), Rojo #C64E4A (destructive), Gris #656565, Negro #000000.
Sidebar oscuro (dark navy). Estilo limpio minimalista. Font: Inter. Idioma: Español.
Lovable Cloud con RLS. Vendedores ven solo sus datos.
